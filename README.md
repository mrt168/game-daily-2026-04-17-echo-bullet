# ECHO BULLET - エコーバレット

跳弾リフレクションシューター。1発の弾が壁で跳ね返り、連鎖爆発を起こしてスコアを稼ぐ30秒タイムアタックゲーム。

## 遊び方
- マウス（タップ）で照準
- クリック（タップ）でショット
- 反射回数が多いほどスコア倍増！

## 技術スタック
- Phaser 3.70
- Vanilla JavaScript
- HTML5 Canvas + Web Audio API

## プレイ
`index.html` をブラウザで開くだけ。または `python3 -m http.server 8099` でローカルサーバー起動。
